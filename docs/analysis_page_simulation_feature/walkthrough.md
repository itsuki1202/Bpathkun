# 修正内容の確認 (Walkthrough): 分析ページの比較・シミュレーション機能

分析ページに、スタッフやチームを比較し、さらに実績の変動を予測できる新機能を追加しました。

## 変更内容メモ

### 1. 「比較・シミュレーション」タブの追加
- 「店舗別分析」「チーム別分析」「項目別分析」に並んで、新しいタブを追加しました。

### 2. 比較機能
- **比較単位**: 「個人」「チーム」「店舗」から選択可能です。
- **対象選択**: AさんとBさん（またはチームAとチームB）を選択すると、合計スコア・順位の差分、および各項目ごとの詳細な比較テーブルが表示されます。

### 3. シミュレーション機能
- **獲得数予測**: 「もしあと●件獲得していたら？」というシミュレーションが可能です。
- **動的再計算**: 入力された件数に基づき、システム全体のランキングと得点をその場で再計算し、「現在の順位より●位アップする」といった具体的な結果を表示します。
- **既存への影響回避**: `calculate_scores` のロジック自体は変更せず、シミュレーション実行時のみメモリ上の仮想データを計算に回すことで、本番データや他ページへの影響をゼロにしています。

## 検証結果

### 動作確認手順
1. 左メニューから「分析」を選択。
2. 「比較・シミュレーション」タブをクリック。
3. 比較単位で「個人」を選択し、任意の2名を選択。
4. 対象Aの「HS新規」などを +5 件にし、「シミュレーションを実行」をクリック。
5. 「順位が●位アップ」と表示されること、および項目別スコアが増加することを確認。

> [!NOTE]
> シミュレーションは「他の全スタッフの成績は変わらない」という前提での予測となります。
