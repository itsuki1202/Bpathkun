# 修正内容の確認 (Walkthrough)

新評価軸「個人達成（individual_achievement）」の実装およびドキュメント更新が完了しました。

## 実施内容

### 1. 新評価軸の実装 (`calculation.py`)
- **内部名**: `individual_achievement`
- **目標値**: マスター設定の「店舗別目標値」（`shop_targets`）の数値を、店舗所属スタッフ全員の「個人目標」として採用します。
- **個人判定**:
    - 分母に関わらず **`個人の実績 ≧ 個人目標`** であれば設定された配点を付与。
    - 未達成の場合は 0 点となります。
- **組織（チーム・店舗）判定**:
    - **組織目標** = `個人目標 × 所属人数` とし、組織の実績合計がこれに達すれば組織スコアを加点します。
    - 例：「A、B、Cさんの3名店舗で個人目標が各2件」の場合、店舗目標は計6件となります。

### 2. Canvasドキュメントの更新
- **「⚖️ 評価軸の種類と計算式」**: ⑤ 個人達成（individual_achievement）の定義を追記。判定条件と組織集計のルールを明示しました。
- **「🔄 スコア計算の全体フロー」**: ⑤の計算ステップに新評価軸を追記。

## 確認方法
1. Canvasの「⚖️ 評価軸の種類と計算式」で新ロジックを確認してください。
2. 以降の計算実行時、評価タイプを `individual_achievement` に設定することで、このロジックが適用されます。
