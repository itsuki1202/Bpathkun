# 実装計画 (Implementation Plan) - Canvasドキュメントの洗練

## 目的
ブライトパスくんの成績計算ロジック（特に分母の扱い）を、コードの実装に基づきCanvas上で正確に定義・文書化する。

## 変更対象
- `ブライトパスくん.canvas`: 
    - 「SAITO式分母」の計算式を修正（基準時間の参照を明示）。
    - 「独自分母」の仕組みを詳細化。

## 実施ステップ
1. **コード分析**: `calculation.py` 等から正確な計算ロジックを抽出。
2. **言語化・合意**: 書き換える前にユーザーに内容を確認。
3. **反映作業**: 
    - Obsidianを閉じてもらい、競合を回避。
    - キャンバスJSONファイルの直接編集。
4. **検証**: ファイル書き換え後の内容確認。
